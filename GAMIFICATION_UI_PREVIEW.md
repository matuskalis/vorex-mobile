## Color Scheme

### Primary Colors
- **Accent/Primary**: `#6366f1` (Indigo) - Buttons, highlights
- **Success**: `#22c55e` (Green) - Completed items, checkmarks
- **Warning**: `#fbbf24` (Amber) - XP, levels
- **Fire/Streak**: `#f97316` (Orange) - Streak indicators
- **Error**: `#ef4444` (Red) - Warnings, errors

### Background Colors
- **Background**: `#0a0a0a` (Near Black)
- **Card**: `#1a1a1a` (Dark Gray)
- **Elevated**: `#262626` (Medium Gray)

### Text Colors
- **Primary**: `#ffffff` (White)
- **Secondary**: `#a3a3a3` (Light Gray)
- **Muted**: `#737373` (Medium Gray)
- **Disabled**: `#525252` (Dark Gray)

### Achievement Category Colors
- **Milestone**: Purple border
- **Streak**: Orange border
- **Practice**: Blue border
- **Mastery**: Gold border
- **Time-based**: Teal border

## Animations

### XPBar
- **Progress Fill**: Spring animation (tension: 40, friction: 8)
- **Duration**: ~300ms for smooth fill
- **Easing**: Natural spring bounce

### StreakBadge
- **Fire Icon**: Subtle pulse effect on active streak
- **Warning Bar**: Fade in when at risk

### AchievementToast
- **Entrance**: 
  - Slide down from -200px to 0
  - Scale from 0.8 to 1.0
  - Fade in opacity 0 to 1
  - All parallel with spring animation
  
- **Exit**:
  - Slide up to -200px
  - Fade out opacity to 0
  - 300ms timing

- **Shine Effect**: Diagonal shine overlay with subtle animation

### AchievementsList
- **Unlock Animation**: Scale bounce when achievement unlocks
- **Filter Change**: Smooth fade transition

## Interactive States

### Pressed State
- Opacity: 0.9
- Scale: 0.98
- Duration: 100ms

### Disabled State
- Opacity: 0.5
- No interaction

### Loading State
- Spinner in center
- Blurred content

## Accessibility

- All text meets WCAG AA contrast requirements
- Touch targets minimum 44x44px
- Screen reader friendly labels
- Reduced motion support

## Responsive Design

**Phone (< 375px):**
- Compact components by default
- Stack vertically
- Smaller font sizes

**Tablet (> 768px):**
- Full components
- Side-by-side layouts
- Larger touch targets
